# Claude Skill — Accessibility Workflow

A collection of Claude skill files built for accessibility practitioners 
working in the TBR system — and anyone using Claude for VPAT reviews, 
vendor assessments, and accessibility compliance workflows.

## What is a Claude Skill?

A Claude skill is a SKILL.md file you load into Claude to give it structured 
guidance for a specific task. Think of it as a reusable prompt template backed 
by real workflow knowledge — so Claude knows not just *what* to do, but *how 
you work*.

## Skills in this Repo

### Accessibility Workflow
Guides Claude through VPAT review, vendor communication, and risk assessment 
tasks aligned with WCAG 2.1 AA and Section 508 standards. Built for the 
TBR context, including DOJ compliance timelines and the AIMT procurement 
process.

**Best for:** Accessibility coordinators, compliance reviewers, instructional 
designers working with vendors

## How to Use a Skill

1. Open the SKILL.md file you want
2. Copy the full contents
3. Paste it into your Claude conversation as a system prompt or at the start 
   of a new chat
4. Claude will follow the workflow guidance in the file

## About

These skills were built during real accessibility compliance work in the 
Tennessee Board of Regents system. They're shared here in the spirit of 
building things that outlast any one person's involvement.

Questions or want to adapt these for your campus? Open an issue or reach out.
